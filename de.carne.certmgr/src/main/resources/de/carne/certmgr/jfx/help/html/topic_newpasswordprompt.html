New password required

This dialog is displayed whenever a new password is required (e.g. during key creation).

The prompt message shows for which object the password is required. The password needs to be entered twice to make avoid unnoticed typos. The dialog will be displayed again if the entered passwords do not match or if the password is rejected due to other reasons. An additional message shows the cause for rejecting the password.

Canceling this dialog will cancel the current operation (e.g. certificate generation).
