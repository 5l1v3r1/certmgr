Neuer Zertifikatspeicher/Zertifikatspeicher-Einstellungen bearbeiten

Dieser Dialog wird sowohl während der Anlage eines neuen Zertifikatspeichers, als auch zur Bearbeitung der Einstellungen des aktuell geöffneten Zertifikatspeichers verwendet.

Name und Speicherort geben gemeinsam den vollstÃ¤ndigen Pfad des Zertifikatspeichers an. Als Speicherort muss ein existierendes Verzeichnis ausgewählt sein. Der Zertifikatspeicher wird dort als Unterverzeichnis mit dem eingegebene Namen angelegt. Entsprechend muss der Name ein gültiger Verzeichnisname sein. Name und Speicherort können aus der Anwendung heraus später nicht mehr geändert werden. Allerdings kann das Zertifikatspeicher-Verzeichnis einfach über die Dateiverwaltung verschoben und umbenannt werden.

Die übrigen Einstellungen legen die Standardwerte fest, welche in der Anwendung z.B. zur Zertifikat-Erzeugung verwendet werden. Diese Standardwerte können in dem jeweiligen Dialog jederzeit angepasst werden.
